Left column: odd pins, Right column: even pins.

## Key Usage Notes (SDR Clock Synchronization)

1. External Clock/PPS Priority: Use J4 first. J4 is pure PL GPIO with no peripheral conflicts.
   Ideal for multi-board clock synchronization.

2. Voltage Level: All PL GPIO are 3.3V LVTTL. External 10MHz clock and PPS signals
   must be 0~3.3V. Overvoltage is prohibited.

3. Multiplexing Rules: J3 pins are default bound to LCD, UART, CAN peripherals.
   To use as GPIO, remove peripheral constraints in FPGA project.

4. Wiring Guidelines: Multi-board synchronization requires common ground.
   Use shielded cables for clock and PPS signals. Keep traces short.
